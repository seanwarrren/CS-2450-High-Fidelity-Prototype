# TripPinner (Frontend Prototype)

A simple collaborative trip planning UI built with HTML, CSS, and JavaScript.

## Features
- Create trips
- View trips in card layout
- Friends list
- Interactive trip map
- Add pins to map
- Vote on locations

## Setup

1. Clone the repo
2. Copy the config example file:
   ```
   cp js/config.example.js js/config.js
   ```
3. Open `js/config.js` and paste your Unsplash Access Key (get one free at https://unsplash.com/developers)
4. Open `index.html` in your browser, or run a local server:
   ```
   python3 -m http.server
   ```
   Then open http://localhost:8000

## Notes
- This is a frontend-only prototype
